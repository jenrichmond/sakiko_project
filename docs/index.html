Coding

art by @allison_horst

Coding 1 - In this exercise, you will wrangle and visualise data from a study testing whether infants associate people with familiar music. Practice making data long, computing summary stats and plotting column graphs with error bars. Ask questions about variability by plotting raw data too. You can read the original paper by Mehr et al., (2016) here.

Coding 2 - Did the Ebola outbreak impact the US election? Do they have enough data in this study to say it did? Practice clean_names, slice, and making scatterplots using geom_point. Learn about the ggpubr package to get correlation coefficients on plots and the naniar package for visualising missing data. You can read the original paper by Beall et al.,here.

Coding 3 - We often overestimate our own knowledge, going as far as claiming knowledge of concepts that don’t exist. Is our tendency to “overclaim” related to how much we think we know about a particular topic? Practice getting summary stats and plotting in this exercise. Use geom_jitter and geom_boxplot to look at variability in scores; geom_point will help with scatter plots. Play with displaying tables in different formats using kableExtra and apaTables. You can read the original paper by Atir et al.here.

Coding 4 - Does shared pain increase group bonding among strangers? Or affect how challenging/threatening we find group activities to be? Practice making data long, computing summary stats, and making clustered column graphs. Also use your wrangling skills to verify that the authors calculated their summary scores correctly. You can read the original paper by Bastian et al. here

MTT project

Coding 5 - woot! we are at the point where we can start working with the MTT dataset. In this exercise, you will read in data for the overgeneral coding and specificity scores, make the data long and answer some questions about group differences using your ggplot skills.

Coding 6 - In this exercise, you will read in data for the phenomenological ratings, make the data long and answer some questions about group differences in the experience of remembering/imagining using your ggplot skills

Coding 7 - Lets use the Tidy Tuesday Taylor Swift and Beyonce analysis as a template for trying sentiment analysis on the narrative responses participants gave in response to past/future, pos/neu/neg events.

Coding 8 - some stats might be useful for your results section. First thing to do is to decide what questions we would like statistical analysis to answer for us about the data. Then we can work out whether we need t-tests, ANOVA, and/or correlation analysis to test hypotheses re group differences in MTT.
